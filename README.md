# Memory-consistent-Autoencoder
The code of memory-consistent autoencoder (MCA).

# Requirements
~~~
pytorch
pandas
scikit-learn
susi
~~~
The code was tested with Pytorch 1.9.0, CUDA 10.2 and Windows 10.  </br>

# Quick start
```shell
python main.py
```
